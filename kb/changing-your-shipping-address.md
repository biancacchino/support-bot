# Changing your shipping address

There are two different things people mean by this, and they work differently.

## Changing the address on a stored address book entry

Sign in, go to **Account settings**, then **Addresses**, and edit the entry.

**This does not affect orders you have already placed.**
The delivery address is copied onto the order at checkout, so an order already in the system keeps the address it was placed with, even after you edit the address book.

This is deliberate.
An order you placed for the old site should still go to the old site, not follow the address book to a new one.

## Redirecting an order you have already placed

Open the order under **Orders** and choose **Change delivery address**.

Whether you can depends on how far it has got:

| Order status | Can you redirect it? |
|---|---|
| Confirmed | Yes, freely, to any address on the account |
| Being picked | Yes, but only to an address in the same delivery region |
| Packed | Sometimes, if the consignment has not been assigned to a vehicle |
| Dispatched | No |

The cut-off for a free change is **4pm the day before delivery**, which is later than the cut-off for editing the order's contents.
You can therefore often redirect an order you can no longer edit.

## Once it is dispatched

We cannot redirect a consignment that is with the courier.

Your options:

- **Contact the courier directly** using the tracking link. Many will hold a consignment at a depot for collection, or redeliver to a different address in the same area, sometimes for a fee. This is between you and them.
- **Refuse the delivery.** It comes back to us and is refunded in full with no cancellation fee. Then reorder to the correct address.

For chilled and frozen goods, refuse rather than attempt a courier redirect.
A perishable consignment that spends an extra day being rerouted will fail its cold chain, and you will end up refusing it anyway, but a day later and with the goods destroyed.

## Changing to a new address, not one already on the account

You can only redirect an order to an address already on the account.

Add the new address first, then redirect.
On credit-terms accounts a new address needs verifying before it can be used, which takes up to 1 working day, so it may not be possible in time for tomorrow's delivery.

## Address changes and delivery cost

Redirecting can change the delivery charge, if the new address is in a different delivery zone.

If it costs more, you are asked to approve the difference before the change takes effect.
If it costs less, the difference is refunded.
Highlands, islands, and offshore addresses are the usual reason a redirect costs more.
